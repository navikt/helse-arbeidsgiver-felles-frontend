# helse-arbeidsgiver-felles-frontend

Felles kode som kan deles mellom frontend applikasjonene for helse arbeidsgiver.

## Slik starter du lokalt

Last ned dependencies 

<pre>
yarn install
</pre>

Kjør koden

<pre>
yarn start
</pre>
